Tenant Commands

/start

Displays the initial welcome greeting.

/start

/help

Displays the configured help text and available commands.

/help

/link <LinkCode>

Connects the tenant's Telegram account to their specific unit using the secure 6-character code generated by the landlord on the dashboard.

/link A1B2C3

/unlink

Disconnects the tenant's Telegram account from their unit. Useful when moving out or switching devices. After unlinking, the tenant must use /link again with a new code to reconnect.

/unlink

/rules

Displays the complete list of Condo House Rules (e.g., swimming pool hours, visitor policies, trash collection).

/rules

/clearance

Outlines the exact steps and requirements for the move-out clearance process.

/clearance

/report <Issue>

Submits a maintenance ticket to the Support Hub. Works for both text messages and photo/video captions.

/report The air conditioning unit is leaking water.

[Attach Photo/Video] + /report

Submits a maintenance ticket with visual evidence. Add /report in the caption to classify it as a concern.

/report Leaking pipe under the sink.

/payment

Submits a payment receipt. Send a photo/video with /payment in the caption. (Note: Sending media with NO caption also defaults to a payment submission).

/payment GCash receipt for March rent

Landlord / Admin Commands Private

These commands will only work for the Telegram Account whose ID matches the OWNER_TELEGRAM_ID registered in the server environment constraints.

/addtenant <UnitNumber> <TenantName> [DueDay]

Manually provisions a new unit and tenant profile via Telegram instead of the dashboard.

/addtenant 110 Noel 15

/removetenant <UnitNumber>

Deletes a unit and disconnects the tenant's data.

/removetenant 110

/tenantlist

Generates a text list of all units, the assigned tenant's name, their due date, and whether they have successfully linked their Telegram account.

/tenantlist
